Plugins JavaScript poderosos sem jQuery

Adicione elementos ocultos comutáveis, modais e menus offcanvas, popovers e tooltips, e muito mais - tudo sem jQuery. O JavaScript do Crow é HTML-first, o que significa que a maioria dos plugins são adicionados com atributos data no seu HTML. Precisa de mais controle? Inclua plugins individuais programaticamente.
