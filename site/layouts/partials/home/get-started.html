Instale via gerenciador de pacotes

Instale os arquivos fonte Sass e JavaScript do Crow via npm, RubyGems, Composer ou Meteor. As instalações gerenciadas por pacotes não incluem documentação ou nossos scripts de compilação completos. Você também pode usar qualquer demonstração do nosso repositório de exemplos para iniciar rapidamente projetos Crow.

{{ highlight (printf ("npm install crow@%s") .Site.Params.current_version) "sh" "" }} {{ highlight (printf ("gem install crow -v %s") .Site.Params.current_ruby_version) "sh" "" }}

Leia nossa documentação de instalação para mais informações e gerenciadores de pacotes adicionais.
