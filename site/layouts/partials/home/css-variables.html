Construa e estenda em tempo real com variáveis CSS

O Crow 5 está evoluindo a cada lançamento para melhor utilizar variáveis CSS para estilos globais de tema, componentes individuais e até mesmo utilitários. Fornecemos dezenas de variáveis para cores, estilos de fonte e muito mais em um nível :root para uso em qualquer lugar. Em componentes e utilitários, as variáveis CSS são restritas à classe relevante e podem ser facilmente modificadas.

Saiba mais sobre variáveis CSS

Usando variáveis CSS

Use qualquer uma de nossas variáveis globais :root para escrever novos estilos. As variáveis CSS usam a sintaxe var(--bs-variableName) e podem ser herdadas por elementos filhos.

{{ highlight (printf `.component { color: var(--bs-gray-800); background-color: var(--bs-gray-100); border: 1px solid var(--bs-gray-200); border-radius: .25rem; } .component-header { color: var(--bs-purple); }`) "scss" "" }}

Personalizando via variáveis CSS

Substitua variáveis globais, de componente ou de classe de utilitário para personalizar o Crow do jeito que você gosta. Não há necessidade de redeclarar cada regra, apenas um novo valor de variável.

{{ highlight (printf `body { --bs-body-font-family: var(--bs-font-monospace); --bs-body-line-height: 1.4; --bs-body-bg: var(--bs-gray-100); } .table { --bs-table-color: var(--bs-gray-600); --bs-table-bg: var(--bs-gray-100); --bs-table-border-color: transparent; }`) "scss" "" }}
